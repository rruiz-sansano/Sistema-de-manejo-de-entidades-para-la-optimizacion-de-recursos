# Sistema-de-manejo-de-entidades-para-la-optimizacion-de-recursos
Trabajo de título enfocado en mostrar las capacidades de Godot para la optimización de recursos.

El objetivo de este trabajo de título es un sistema de 4 experiencias que muestran una implementación de una forma de optimización cada una, para ser utilizada como herramienta de aprendizaje. En el archivo "Memoria.pdf" se encuentra toda la información sobre este proyecto y cómo fue construido.

Para revisar el proyecto es necesario tener Godots 4.5 (a menos que haya una actualización importante debería ser compatible con versiones posteriores). Luego, basta con importar este repositorio y abrir el proyecto.

Se incluye el ejecutable del proyecto, llamado "test.exe", para su fácil acceso.
